Image Formats

Lossy vs. Lossless

Format Compression Type
JPEG Lossy
PNG Lossless
GIF Lossless (indexed color, up to 256 colors)
WebP Offers both lossy and lossless modes
BMP Typically uncompressed (or RLE-based, lossless)
TIFF Often lossless (supports optional JPEG-based lossy)
SVG Vector-based (not raster-based compression; can be gzipped but not “lossy” in the same sense)

Animation Support

Format Animation Support
JPEG No
PNG No (though APNG is an extension that supports animation)
GIF Yes
WebP Yes (in animated WebP)
BMP No
TIFF Yes (multi-page TIFF can be used for basic animation, though not common on the web)
SVG Yes (through SMIL animations or CSS/JavaScript; not a “frame-by-frame” raster animation)

Transparency Support

Format Transparency Support
JPEG No (no alpha channel)
PNG Yes (full alpha channel)
GIF Yes (1-bit transparency; on/off only)
WebP Yes (supports alpha channel in lossless & lossy modes)
BMP Not typically (standard BMP has no alpha channel)
TIFF Yes (supports alpha channel in many implementations)
SVG Yes (vector shapes can be fully or partially transparent)

Raster vs Vector

Format Raster or Vector
JPEG Raster
PNG Raster
GIF Raster
WebP Raster
BMP Raster
TIFF Raster
SVG Raster

Notes

Lossy vs. Lossless:
  • Lossy sacrifices exact detail for smaller file size (e.g., JPEG).
  • Lossless retains all original data when compressed (e.g., PNG, GIF).
Animation:
  • GIF and WebP are commonly used for web animation.
  • PNG’s APNG extension allows animation, but it’s less universally supported.
  • TIFF supports multi-page images, but it’s rarely used for actual animations online.
  • SVG can be animated using markup/CSS but is fundamentally different from frame-based raster animation.
Transparency:
  • PNG and WebP offer full alpha-channel transparency.
  • GIF supports only on/off transparency for one color.
  • JPEG does not support transparency.
Raster vs. Vector:
  • Raster images are defined by a grid of pixels—scaling too large can pixelate or blur them.
  • Vector images are defined by mathematical shapes/paths—infinitely scalable without quality loss.
